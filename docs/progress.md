> **更新规则（强制）**：
> 1. 代码模块完成时，将状态改为 ✅，填写 `完成` 时间（精确到时分秒）和 `提交` 列的 commit hash。

## 开发阶段总览

```
Phase 0 — 对话问答 MVP（含行业知识库）
  ├─ 后端: ORM → RAG检索(P0) → G代码引擎 → /api/chat/send → 加工归档骨架
  └─ 前端: 对话收发 → 图片/语音输入 → 标签引用UI → 离线缓存策略(P2)

Phase 1 — G代码生成核心链路（含四层安全体系）
  ├─ 后端: G代码生成API → 后处理模板 → 图纸解析API → 首检/刀具API → L4审批API
  └─ 前端: 步骤1（上传+质检+下拉）→ 步骤2（解析+置信度+编辑）→ 步骤3（代码+内联+确认）→ 步骤4（首检+刀具+档案）
```
